About Tiberius James

I am a software engineer. I got my start in tech back in 2008 as tech support for a local cable internet company. After earning my Associate's at Vatterott College in Computer Systems, I began honing my craft. From hardware troubleshooting to software installs, I wore many hats: Computer Technician, Network Administrator, Team Lead, and Fixer-of-Many-Things.

I later transitioned into customer service roles in the insurance and banking industries—because soft skills are essential when complex information needs to be communicated clearly. In 2016, I returned to college at Colorado State University - Global to pursue a Bachelor's degree. Though I earned a B.S. in Criminal Justice, I developed valuable skills applicable to software development: communication, critical thinking, ethics, and analysis.

In 2025, I enrolled at Hack Reactor's Beginner’s Bootcamp for Software Engineering. I’ve since gained experience in backend languages like Python, PostgreSQL, and FastAPI, as well as frontend tools like JavaScript, React, HTML/CSS, and DOM manipulation.

Additional proficiencies include debugging, troubleshooting, code review, pair programming, Agile development, object-oriented programming, version control with Git, Docker, and using productivity tools like Google Docs and Microsoft Office.
