About Tiberius James



I am a software engineer. I got my start in tech back in 2008 where I started out as tech support for a local cable internet company. After I earned my Associates at Vatterott College in Computer Systems, I set out to learn my craft. From hardware troubleshooting to software installs, I wore many hats: Computer Technician, Network Administrator, Technician, Team Lead, Fixer-of-many-things.

I then transitioned to the world of Customer Service in the Insurance, and Banking industries because soft skills are important when complicated information needs to be broken down into terms that anyone can understand. In 2016, I went back to college at Colorado State University - Global to pursue my Bachelor's Degree. Although a different path was taken with the BS in Criminal Justice, I learned many skills which translate to software development including communication, critical thinking, ethics, and analysis. Later, in 2025 I enrolled at Hack Reactor's Beginner's Bootcamp for Software Engineering.

I've learned backend programming languages such as Python, PostgreSQL, and FastAPI. With frontend I learned JavaScript, React, HTML/CSS, and DOM Manipulation. Some software engineering technologies and developments include: Debugging, Troubleshooting, Code Review, Pair Programming, Agile Development, and Object Oriented Programming. Lastly, DevOps and Platforms: Git, Docker, Google Docs, and Microsoft Office.
